R-ArcGIS Bridge

R-ArcGIS Bridge

Compatibility & Installation

Requires a licensed ArcGIS desktop version

The R-ArcGIS Bridge works for

  • ArcPro 2.0 (native support)
    • Install from ArcPro
    • Instruction Video
  • ArcMap 10.3.1 / ArcPro 1.1
    • Instructions on GitHub

What You Can Do with the Bridge

  • View metadata before bringing spatial info into R.
  • Import spatial layers from disk, or file geodatabases.
  • Create R scripts that you run from ArcGIS.

Example

library(arcgisbinding)
library(sp)
arc.check_product()
## product: ArcGIS Pro ( 12.1.0.10257 )
## license: Advanced
## Create the connection
crimepts_con <- arc.open(path="../exercises/data/SF_Crime.gdb/San_Francisco_Crimes")

crimepts_adf <- arc.select(crimepts_con)

## [11] "Y"

## [1] 74760

Convert to a sp object.

crimepts_spdf <- arc.data2sp(crimepts_adf)
class(crimepts_spdf)
## [1] "SpatialPointsDataFrame"
## attr(,"package")
## [1] "sp"

Plot.

plot(crimepts_spdf, axes=TRUE, pch=16, cex=0.3)
