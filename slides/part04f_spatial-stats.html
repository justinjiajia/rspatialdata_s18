What exactly is Spatial Data Analysis?

  • using location or spatial relationships as an explanatory or predictive variable
  • modeling the process that produced the pattern
  • examining the effects of scale
  • backdrops to show context

Common tasks working with geospatial data

Data Prep

  • Import and export
  • Convert between formats
  • Project

  • Create new data
  • Query by attribute or spatial property
  • Join by attribute or spatial relationship (fuse)
  • Geocode

Analysis

  • Measure geometric properties
    • length
    • area
    • perimeter
    • sinuosity
    • texture
  • Spatial stats
  • Interpolation
  • Geoprocessing
  • Compare (siting)
  • Spatial sampling
  • Scale dependency
  • Classify
  • Model process (e.g., regression)
