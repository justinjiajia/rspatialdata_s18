Filtering with Square Brackets

Option 1: pass a vector of integers:

x <- 4:6  
idx <- c(1,2)
x[idx]
## [1] 4 5

Option 2: pass a vector of Booleans:

idx <- c(T,T,F)
x[idx]
## [1] 4 5

Option 3: pass an expression or function which returns a vector of Booleans:

x > 4
## [1] FALSE  TRUE  TRUE
x[ x > 4 ]
## [1] 5 6
