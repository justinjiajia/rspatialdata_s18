Spatial Analysis with R:
Exercise 2: Finding and Installing Packages

Spring 2018

The key to working effectively in R is finding and installing the right packages. For spatial data, some of the most common and useful packages include:

  • sp: classes and methods for spatial data
  • raster: classes and methods for common raster data operations
  • rgdal: importing and transforming different spatial data types and providing access to proj4 library for working with map projections and coordinate systems

############################
#### EXPLORING PACKAGES ####
############################

# 1. Load the sp, raster, and rgdal packages.

library(sp)
library(raster)
library(rgdal)

# 2. Run the 'gallery' demo from the sp package.

The package must be loaded before you can run the demo() function. 
library(sp)  
demo(gallery)

# 3. What datasets are bundled with the maps package?

install.packages("maps")
data(package="maps")

# 4. What type of object is the 'world.cities' from the maps package?

You may need to install the maps package if you haven't already. 
library(maps)
class(world.cities)
## [1] "data.frame"
